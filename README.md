# ninjarun2D
leaning_basicunity
ผลงานตัวนี้คือการเรียนรู้การทำเกมยูนิตสองมิติและการเชื่อมต่อกับกิตฮับ 
เกมที่สร้างเป็นเกมที่พี่ kongruksiam สอนในyoutube ชื่อ ninjarunner ผมทำเพื่อศึกษาการเขียนโค้ดและทำเกมบนunity
โดยพี่kongruksiam จะสอนระบบพื้นฐานต่างๆ,ไม่ว่าจะเป็น,การเปลี่ยนฉาก,ระบบการเคลื่อนไหว,หลักการทำanimationของตัวละคร,ระบบฟิสิก,hitboxตัวละครและวัตถุ ฯลฯ

เมื่อผมเรียนพื้นฐานจากคลิปวีดิโอเสร็จก็ไปค้นคว้าหา scripst หรือฟังก์ชันที่ผมสนใจ ไม่ว่าจะเป็น การกระโดด 2 ชั้น เหยียบหัวศัตรูแล้วตาย อมตะเป็นเวลา1.5วิ ระบบจำนวนพลังชีวิต
ซึ่งฟังก์ชันทั้งหมดผมเล่นรู้จาก Ai ที่ช่วยแก้ไขและอธิบายcodeที่ผิดพลาดและทำให้ผมสร้างเกมNinjaออกมาได้
